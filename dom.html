Pendahuluan Document Object Model (BOM)

Cara JavaScript Mengontrol Website dan Browser Setelah membaca materi sebelumnya, maka akan muncul pertanyaan “Bagaimana JavaScript mengontrol sebuah website atau browser?” Jawabannya ada dua, dan keduanya akan kita bahas di modul ini, yakni melalui Browser Object Model (BOM) dan Document Object Model (DOM). Yang pertama adalah BOM. Dengan BOM, kita dapat memberikan perintah-perintah khusus ke browser, misalnya melalui sebuah 'atribut' khusus milik browser yakni 'window' (akan kita bahas di materi selanjutnya) sehingga kita bisa membuat browser menampilkan pesan pop-up.
